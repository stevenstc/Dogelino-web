Comandos para ejecutar en el Bot

Recuerda que debes de tener descargada la aplicación Telegram para acceder al Bot

Comenzando

/Start: inicia el Bot registrando tu cuenta de telegram en nustras bases de datos

Datos basicos y de estado

/perfil: muestra tus datos basicos

  • Usuario
  • Fecha en la que te registraste
  • Dirección Dogecoin
  • El saldo vinculado a esa Dirección Dogecoin
  • Tus estados de DTF

Información de saldo vinculado

/saldo: muestra tu saldo +

  • /nuevoPagoHash: reclamar tarjeta Hash de regalo
  • /nuevoRetiro: crea una orden para enviar tus DOGE a tu cartera y poder utilizarlos fuera de la aplicación
  • /misRetiros: muestra un historial de todos los retiros realizados ordenados desde la aplicación

NOTA: Las opciones se actualizan segun vas usando la aplicación.

Información de cual será la utilización de tus DOGE

/invertir: muestra tus posibilidades

  • /misDTF: muestra un historial de todos los DTF realizados ordenados desde la aplicación
  • /abrirDTF: crea una orden para enviar tus DOGE a un inversor que gestiona tu saldo para generar rentabilidad abriendo un DTF*
  • /reclamarDTF: utilizala para reclamar los DTF listos para reclamar cuando se haya cumplido su plazo

NOTA: Las opciones se actualizan segun vas usando la aplicación.

DTF* es un Deposito a Termino Fijo
